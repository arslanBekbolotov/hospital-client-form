# Форма Создания Клиента

Используемые технологии:
• Vue.js (Vue2)
• Vuelidate
• Scss

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```
